Problem 8

Category: N/A

Agnes writes the following four statements on a blank piece of paper.

  • $\bullet$ At least one of these statements is true.
  • $\bullet$ At least two of these statements are true.
  • $\bullet$ At least two of these statements are false.
  • $\bullet$ At least one of these statements is false.

Each statement is either true or false. How many false statements did Agnes write on the paper?

$$\textbf{(A) } 0 \qquad\textbf{(B) } 1 \qquad\textbf{(C) } 2 \qquad\textbf{(D) } 3 \qquad\textbf{(E) } 4$$
Difficulty: 13/100 — A clean logic puzzle; a step down from the previous misplaced early problems.
Core Concepts: logic
Challenges: Interpreting what it really means if a statement is true or false. Recognize that the statements can be implied off each other (i.e., statement 2 being true requires statement 1 to be true too.)
Techniques: Test each scenario by number of true statements.
Error-prone Steps: Extracting the incorrect result, or failing to understand the problem.
Ideal Time:
Experienced: ≤ 1 min
Intermediate: ≤ 1 min
Beginner: ≤ 1.5-2 min
