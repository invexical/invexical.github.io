Problem 9

Category: Algebra

Let $f(x)=100x^3-300x^2+200x$. For how many real numbers $a$ does the graph of $y=f(x-a)$ pass through the point $(1,25)$?

$$\textbf{(A) } 1 \qquad \textbf{(B) } 2 \qquad \textbf{(C) } 3 \qquad \textbf{(D) } 4 \qquad \textbf{(E) } \text{more than } 4$$
Difficulty: 51/100 — Horizontal shifts, graph shape, number of roots.
Core Concepts: Understand $f(x-a)$; analyze graph without solving exact equation.
Techniques: Calculate maxima and minima to count intersections.
Error-prone Steps: Attempting exact roots unnecessarily.
Ideal Time:
Experienced: ≤ 2-3 min
Intermediate: ≤ 4 min
Beginner: ≤ 10 min
